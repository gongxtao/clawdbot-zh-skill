# 节日查询技能

## 使用方法

直接用自然语言：
```
"2026年春节是哪天"
"明天是什么节日"
"国庆节放几天假"
"清明节放假安排"
```

## 功能说明

- **节日查询**：农历/公历节日查询
- **放假安排**：节假日放假天数
- **调休安排**：周末调休上班日
- **节日倒计时**：距离节日还有多少天
