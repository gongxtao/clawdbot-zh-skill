# 星座运势技能

## 使用方法

直接用自然语言：
```
"今天运势怎么样"
"天秤座今日运势"
"狮子座性格分析"
"双子和天蝎座配不配"
```

## 支持的星座

白羊座、金牛座、双子座、巨蟹座、狮子座、处女座、天秤座、天蝎座、射手座、摩羯座、水瓶座、双鱼座

## 功能说明

- **每日运势**：综合运势、爱情、事业、财运
- **性格分析**：性格特点、优缺点
- **星座配对**：两个星座的匹配度
