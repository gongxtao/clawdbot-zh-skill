# 空气质量查询技能

## 使用方法

直接用自然语言：
```
"北京空气质量"
"上海 PM2.5"
"今天空气质量怎么样"
"广州空气质量指数"
```

## 功能说明

- **AQI 指数**：空气质量综合指数
- **PM2.5/PM10**：细颗粒物浓度
- **首要污染物**：主要污染来源
- **健康建议**：户外活动建议

## 支持城市

支持全国主要城市空气质量查询。
