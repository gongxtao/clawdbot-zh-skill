# 垃圾分类查询技能

## 使用方法

直接用自然语言：
```
"电池是什么垃圾"
"外卖盒怎么分类"
"过期药品怎么处理"
"快递纸箱是什么垃圾"
```

## 垃圾分类

- **可回收物**：纸类、塑料、金属、玻璃、织物
- **有害垃圾**：废电池、废灯管、废药品、废油漆
- **厨余垃圾**：剩菜剩饭、瓜皮果核、蛋壳、茶渣
- **其他垃圾**：卫生纸、纸尿裤、烟蒂、污损塑料
