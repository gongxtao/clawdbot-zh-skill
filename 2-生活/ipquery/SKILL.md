# IP 查询技能

## 使用方法

直接用自然语言查询：
```
"我的IP"
"查询 IP 1.2.3.4"
"IP 归属地"
"IP 运营商"
```

## 功能说明

- **本机 IP**：查询当前设备的公网 IP
- **IP 信息**：查询指定 IP 的归属地、运营商
- **批量查询**：支持多个 IP 查询

## API 说明

使用 ip-api.com 等免费 IP 查询接口。
