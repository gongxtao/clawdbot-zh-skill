# 农历转换技能

## 使用方法

直接用自然语言查询：
```
"今天农历多少"
"明天农历几号"
"2026年春节是哪天"
"2026年1月1日农历"
"今天是什么节气"
"寒露是哪天"
"2026年国庆节"
```

## 功能说明

- **公历转农历**：查询指定日期对应的农历
- **农历转公历**：查询农历日期对应的公历
- **节假日查询**：春节/国庆/清明/中秋等
- **二十四节气**：立春/雨水/惊蛰/春分等

## API 说明

使用内置农历数据，支持 1900-2100 年。
