# 新闻资讯技能

## 使用方法

直接用自然语言查询：
```
"今天的新闻"
"科技新闻"
"财经资讯"
"有什么热点"
"微博热搜"
"知乎热榜"
```

## 新闻分类

- **今日新闻**：综合头条
- **科技新闻**：科技数码互联网
- **财经资讯**：股市宏观经济
- **体育新闻**：足球篮球赛事
- **娱乐新闻**：影视明星八卦
- **热搜榜单**：微博/知乎/抖音热榜

## API 说明

使用今日头条/百度/新榜免费接口。
