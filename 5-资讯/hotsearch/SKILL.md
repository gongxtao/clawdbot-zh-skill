# 热搜榜单技能

## 使用方法

直接用自然语言查询：
```
"微博热搜"
"知乎热榜"
"抖音热搜"
"B站热门"
"有什么热点"
"热搜榜"
```

## 支持的平台

- **微博热搜**：实时热搜榜
- **知乎热榜**：知乎热门问题
- **抖音热搜**：抖音热点榜
- **B站热门**：Bilibili 热门视频
- **小红书**：热门笔记
- **百度热搜**：百度实时热搜

## API 说明

使用各平台公开接口或第三方聚合 API。
