# 天气预警技能

## 使用方法

直接用自然语言：
```
"天气预警"
"台风预警"
"暴雨橙色预警"
"有没有极端天气"
```

## 预警类型

- **台风预警**：台风路径、等级
- **暴雨预警**：蓝色/黄色/橙色/红色
- **暴雪预警**：积雪、道路结冰
- **高温预警**：中暑、热射病提醒
