# 音乐播放技能

## 使用方法

直接用自然语言：
```
"播放周杰伦的稻香"
"网易云热门歌单"
"我喜欢的音乐"
"播放列表"
"暂停播放"
```

## 功能说明

- **歌曲搜索**：按歌名/歌手/歌词搜索
- **歌单推荐**：热门歌单、推荐歌单
- **播放控制**：播放、暂停、下一首
- **我的音乐**：我喜欢的音乐、播放历史

## API 说明

使用网易云音乐/QQ音乐开放 API。
