# 电影票查询技能

## 使用方法

直接用自然语言查询：
```
"今天有什么电影"
"北京电影票"
"流浪地球3场次"
"最近评分高的电影"
```

## 功能说明

- **热映查询**：当前热映电影
- **场次查询**：影院、场次、票价
- **评分查询**：豆瓣/猫眼评分
- **即将上映**：即将上映的电影

## API 说明

使用猫眼/淘票票开放 API。
