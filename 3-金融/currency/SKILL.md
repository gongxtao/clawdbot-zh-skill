# 汇率查询技能

## 使用方法

直接用自然语言查询：
```
"1美元等于多少人民币"
"欧元汇率"
"100日元换人民币"
"英镑对人民币汇率"
"今日美元走势"
```

## 支持的货币

- **美元**（USD）：美元、美金
- **人民币**（CNY）：人民币、元、块
- **欧元**（EUR）：欧元、欧罗
- **英镑**（GBP）：英镑、磅
- **日元**（JPY）：日元、日圆
- **港币**（HKD）：港币、港币
- **澳元**（AUD）：澳元
- **加元**（CAD）：加元
- **瑞郎**（CHF）：瑞郎、瑞士法郎

## API 说明

使用外汇 API 或银行汇率接口。
