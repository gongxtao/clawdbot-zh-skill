# 加密货币查询技能

## 使用方法

直接用自然语言查询：
```
"比特币价格"
"ETH 行情"
"BTC 涨了多少"
"以太坊市值"
"今日加密货币走势"
```

## 支持的币种

- **BTC**（Bitcoin）：比特币、中本聪币
- **ETH**（Ethereum）：以太坊、V 神币
- **BNB**：币安币
- **SOL**（Solana）：索拉纳
- **XRP**（Ripple）：瑞波币
- **ADA**（Cardano）：艾达币
- **DOGE**（Dogecoin）：狗币
- **MATIC**（Polygon）：马蹄币

## API 说明

使用 CoinGecko 免费 API 或 Binance API。
