# 基金查询技能

## 使用方法

直接用自然语言查询：
```
"易方达蓝筹净值"
"基金代码161039"
"我的基金收益"
"定投基金推荐"
```

## 功能说明

- **净值查询**：基金最新净值、日增长率
- **收益率查询**：近一周/月/年的收益
- **持仓查询**：基金重仓股
- **定投推荐**：热门定投基金

## API 说明

使用天天基金/蛋卷基金开放 API。
