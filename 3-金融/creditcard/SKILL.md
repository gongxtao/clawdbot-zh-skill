# 信用卡账单查询技能

## 使用方法

直接用自然语言：
```
"信用卡账单"
"什么时候还款"
"信用卡积分"
"本月信用卡还款金额"
```

## 功能说明

- **账单查询**：本月账单金额
- **还款日**：还款到期日期
- **还款金额**：应还金额、最低还款
- **积分查询**：信用卡积分余额
