# 股票查询技能

## 使用方法

直接用自然语言查询：
```
"贵州茅台股价"
"腾讯股票"
"苹果股票价格"
"上证指数"
"深证成指"
"恒生指数"
```

## 股票代码识别

- **A 股**：支持拼音/中文（贵州茅台、五粮液、平安银行）
- **港股**：支持腾讯、美团、阿里巴巴等
- **美股**：支持 Apple、Google、Microsoft、Tesla 等

## API 说明

使用东方财富/同花顺免费接口，部分数据需要 API Key。
