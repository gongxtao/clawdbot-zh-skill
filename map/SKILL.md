# 地图定位技能

## 使用方法

直接用自然语言查询：
```
"附近的餐厅"
"距离天安门有多远"
"北京大学在哪里"
"从上海到北京有多远"
"附近加油站"
```

## 功能说明

- **周边搜索**：餐厅、酒店、加油站、ATM等
- **距离计算**：两点间距离测量
- **位置查询**：地标/地址定位
- **路线规划**：简单的路线建议

## API 说明

使用高德地图/百度地图 API，需要配置 API Key。
