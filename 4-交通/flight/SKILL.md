# 机票查询技能

## 使用方法

直接用自然语言：
```
"北京到上海的机票"
"明天北京飞广州"
"查询航班 CA1234"
"今天北京到深圳哪个便宜"
```

## 功能说明

- **航班查询**：按线路查询航班
- **票价查询**：经济舱/商务舱/头等舱
- **航班状态**：准点/延误/取消
- **机场信息**：出发/到达机场
