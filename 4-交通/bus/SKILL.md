# 公交地铁查询技能

## 使用方法

直接用自然语言查询：
```
"北京地铁1号线"
"上海公交911路"
"从望京到国贸怎么坐车"
"最近的地铁站在哪"
"某某路公交车时刻表"
```

## 功能说明

- **线路查询**：公交/地铁线路信息
- **到站查询**：实时到站时间
- **换乘规划**：起点→终点路线
- **附近站点**：周边公交/地铁站

## API 说明

使用高德地图/百度地图公交 API。
