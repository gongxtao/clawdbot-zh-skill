# 火车票查询技能

## 使用方法

直接用自然语言查询：
```
"北京到上海的高铁"
"明天北京到广州的火车"
"2026年2月15日上海到深圳"
"查询车次 G1234"
```

## 功能说明

- **车次查询**：按线路查询车次
- **余票查询**：实时余票状态
- **票价查询**：各座位票价
- **时刻表**：到站/发车时间

## API 说明

使用 12306 API 或第三方火车票 API。
