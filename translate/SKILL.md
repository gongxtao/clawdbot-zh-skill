# 翻译技能

## 使用方法

直接用自然语言翻译：
```
"翻译 Hello World"
"把你好翻译成英文"
"I love China 中文"
"请把下雨了翻译成日文"
"How are you 日语"
```

## 支持语言

- 中文 ↔ 英文
- 中文 ↔ 日文
- 中文 ↔ 韩文
- 中文 ↔ 法文
- 中文 ↔ 德文
- 英文 ↔ 其他语言

## API 说明

使用百度翻译/有道翻译 API，免费版有调用限制。
