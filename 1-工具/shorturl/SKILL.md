# 短链接生成技能

## 使用方法

直接用自然语言：
```
"生成短链接 https://example.com/very/long/url"
"短链接 http://example.com"
"压缩 https://xxx.com/xxxxx"
```

## 功能说明

- **长链接转短链接**：支持主流短链接服务
- **短链接还原**：查询短链接原始地址
- **批量生成**：支持多个链接

## 支持的服务

- 短链接：T.cn、sina.lt、suo.im 等
