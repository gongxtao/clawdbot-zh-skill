# 二维码生成技能

## 使用方法

直接用自然语言：
```
"生成二维码 https://example.com"
"二维码 hello world"
"生成名片二维码 张三 13800138000"
"把我的微信二维码生成出来"
```

## 功能说明

- **链接转二维码**：网址转二维码图片
- **文本转二维码**：任意文本转二维码
- **名片二维码**：姓名+电话+邮箱
- **微信/支付宝二维码**
