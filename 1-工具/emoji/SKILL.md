# 表情搜索技能

## 使用方法

直接用自然语言：
```
"开心的表情"
"生气的emoji"
"赞的表情包"
"爱心 emoji"
```

## 功能说明

- **关键词搜索**：根据语义搜索表情
- **分类浏览**：按心情/事物分类
- **复制使用**：一键复制表情

## 支持的搜索

- 心情：开心、难过、生气、惊讶、害怕
- 动作：点赞、鼓掌、握手、拥抱
- 事物：爱心、星星、太阳、月亮
- 动物：猫咪、狗狗、猪猪
