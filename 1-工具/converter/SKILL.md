# 单位换算技能

## 使用方法

直接用自然语言换算：
```
"100厘米等于多少米"
"1公里等于多少英里"
"37度等于多少摄氏度"
"1公斤等于多少斤"
"100美元等于多少人民币"
"1小时等于多少秒"
```

## 支持的换算类型

- **长度**：米、厘米、毫米、公里、英里、英尺、英寸
- **重量**：千克、克、毫克、斤、公斤、磅、盎司
- **体积**：毫升、升、加仑、夸脱
- **面积**：平方米、平方厘米、平方公里、公顷、亩
- **温度**：摄氏度、华氏度、开尔文
- **速度**：米/秒、公里/小时、英里/小时、节
- **时间**：秒、分钟、小时、天、周、月、年
- **货币**：美元、人民币、日元、欧元、英镑、港币

## API 说明

使用固定换算比率，支持常用单位。
