# 身份证查询技能

## 使用方法

直接用自然语言查询：
```
"身份证号310101199001011234归属地"
"123456199001011234是哪里的"
"身份证校验110101200001011234"
```

## 功能说明

- **归属地查询**：身份证前6位对应地区
- **生日提取**：从身份证号提取出生日期
- **性别判断**：根据号码判断性别
- **校验验证**：校验身份证格式是否正确

## 数据说明

使用身份证号段公开数据库，无需 API。
