# 快递查询技能

## 使用方法

直接用自然语言查询：
```
"查询快递单号 SF1234567890"
"我的顺丰快递到哪了"
"申运单号 8888888888"
"帮我看看中通快递"
```

支持公司自动识别，无需手动指定。

## 快递公司代码

自动识别以下公司：
- 顺丰：SF
- 申通：STO
- 圆通：YTO
- 中通：ZTO
- 韵达：YD
- 京东：JD
- 德邦：DB
- 邮政：YZ
- 天天：TT
- 极兔：JT

## API 说明

使用快递100免费接口，无需注册 API Key。
